# FoodSaver 🍏

FoodSaver is a premium, modern, and highly responsive web application designed to reduce food waste by connecting Restaurants/Supermarkets/Bakeries (Donors), Charities, and Volunteers.

## Tech Stack
- **Backend**: Node.js, Express, Session-based RBAC Authentication
- **Frontend**: Single Page Application structure with native Javascript, HTML5, and Vanilla CSS styling
- **Charts**: Chart.js for interactive analytics dashboards
- **AI Engine**: Advanced simulated forecasting for food waste prediction and proximity matching

## Portals & Roles
- **Donor Dashboard (Restaurant/Bakery/Supermarket)**: Add food donations, view analytics, manage requests, track deliveries, and consult AI prediction tools.
- **Charity Dashboard**: Browse available donations, request surplus food, chat with donors/couriers, and review historical impact.
- **Volunteer Dashboard**: Claim open deliveries, view route mapping, update pickup/delivery steps, and confirm delivery completion.
- **Admin Control Panel**: Full platform insights, user management, donation lifecycles, active dispatches, and system reports auditing.

## Setup & Running
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the application:
   ```bash
   npm start
   ```
3. Open in your browser: `http://localhost:3000`
